# 迎接小生命

> 写在孩子出生前。

## 准爸爸的笨拙清单

把要准备的事按"孩子出生前 / 月子期 / 0-1 个月"三个阶段切开，分别列清单——这是写代码养成的本能。

### 出生前（待产包 + 心理建设）

- [ ] 待产包：妈妈 + 宝宝两套，按医院清单核对
- [ ] 月嫂 / 月子中心：提前 3 个月预约，比想象中紧张
- [ ] 新生儿用品：别囤太多，奶粉先少量试，纸尿裤别买大牌大包
- [ ] 家里清洁 + 空气净化器换滤芯
- [ ] 跟领导报备好陪产假

### 月子期（妈妈优先）

- [ ] 母乳 vs 奶粉：先尊重妈妈意愿，别自己代言
- [ ] 产后情绪：留意妈妈状态，不要把"育儿任务"全压在她身上
- [ ] 双方父母的边界：提前商量好谁来帮忙、待多久、规矩谁定

### 0-1 个月（手忙脚乱期）

- [ ] 喂奶 / 拍嗝 / 换尿不湿三件套，老公必须会
- [ ] 黄疸 / 红屁股 / 肠绞痛——三个最常见小麻烦的应对
- [ ] 夜班排班：父母不能两个人一起熬，要轮班
- [ ] 别天天朋友圈晒娃

## 心理建设

孩子出生这件事，**最大的变化不是多了一个人，而是夫妻关系从"两人协作"变成"三人协作 + 一个完全不会沟通的成员"**。

预期管理：
- 头 3 个月就是熬。睡眠不足是常态。
- 任何育儿建议都不一定对，要自己观察孩子。
- 老人会有自己的方式，不可能 100% 跟你一致。挑大事坚持，小事让步。

## 写下来的意义

我希望以后孩子长大了，能看到爸爸是怎么紧张又笨拙地准备迎接他/她的。

也希望几年后再翻这篇，能笑话当年的自己想多了。
