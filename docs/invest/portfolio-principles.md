# 我的持仓与建仓原则

> 不追求暴富，只追求"睡得着觉"。

## 核心纪律

| 项目 | 上限 | 说明 |
|---|---|---|
| 单只个股仓位 | ≤ 5% | 集中爆仓概率不能影响家庭现金流 |
| 总仓位 | ≤ 50% | 留一半子弹应对极端行情 |
| 单笔止损 | ≤ -5% | 不抗，错了就承认 |
| 不追涨停股 | — | 流动性陷阱风险高 |

## 选股偏好

- **底仓防守优先**：低位 + 回购 + 高股息
- **价值 + 长线** 为主，**波段做 T** 为辅
- 重点关注：半导体、创业板低估值龙头
- 偏好：低估值高股息标的

## 建仓节奏：分档建仓

| 档位 | 触发条件 | 占目标仓位 |
|---|---|---|
| 第 1 档 | 跌至预设买点 | 30% |
| 第 2 档 | 跌幅再 -5% 或缩量企稳 | 30% |
| 第 3 档 | 主力流入 + 板块共振 | 40% |

## 当前持仓骨架

::: tip
具体股票代码与买入价位为个人记录，不在此博客公开披露。
:::

整体配置思路：
- **消费 + 医疗** 防守底仓（稳定现金流）
- **半导体 + 高端制造** 进攻仓（弹性）
- **小市值成长股** 少量观察仓（学习用）

## 复盘原则

每次交易后写复盘，重点记录：
1. 决策时的核心理由
2. 实际走势与预期偏差
3. 错在哪里（情绪 / 数据 / 时机）
4. 下次改进点
