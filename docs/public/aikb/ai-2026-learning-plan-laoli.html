① 全景：6 阶段从 0 到能上岸（含复习锚点）

0
地基期 · Python 生态 + 大模型初体验 约 2–3 周
你不需要从 0 学 Python，目标是「够用」：能流畅调 API、处理数据、写脚本。一周内跑通第一个 LLM 调用。
  • Python 速通：语法差异速查（对照 Java）、虚拟环境、NumPy/Pandas、requests/httpx
  • 大模型调用：OpenAI / Qwen / DeepSeek API、流式输出、Function Calling、结构化输出（JSON mode）
  • Java 选手专属：Spring AI / LangChain4j 入门（你的主栈不丢）
  • 📌 复习锚点：python-for-java-devs · python-env-setup
1
原理期 · 大模型「够用级」原理 约 3 周
不深挖数学，但面试一面常问「对 AIGC 的理解」「了解哪些大模型」。要能讲清 Transformer/Attention 的直觉和 trade-off。
  • 必懂：Tokenizer/Embedding、Attention、Transformer、预训练 vs 微调、上下文窗口、幻觉成因
  • 2026 新增热点：推理模型（o1/o3/R1/QwQ）、Test-Time Compute、MoE、长上下文
  • 📌 复习锚点：transformer-architecture · attention-mechanism · reasoning-models-and-test-time-compute
2
RAG 期 · 检索增强（面试必问，深挖 trade-off）约 4 周 ★重点
2026 面试不再问「RAG 流程是什么」，而追问每个环节你做了什么优化。这是你审核知识库经验的最佳变现点。
  • 全链路：文档解析 → 切块策略（语义切分 vs 固定长度）→ Embedding 选型 → 向量库 → 混合检索（向量+BM25+RRF）→ Rerank → 生成引用标注
  • 深水区：Lost in the Middle、Query Rewrite、知识库更新防旧索引污染、检索质量 vs 答案质量评估、元数据过滤 / 权限隔离
  • 进阶范式：Self-RAG、GraphRAG、自适应检索
  • 📌 复习锚点：rag-architecture · vector-database · embedding-and-encoding
3
Agent 期 · 智能体编排（2026 最热）约 4 周 ★重点
字节/快手/腾讯 Agent 岗高频。核心判断标准：多步自主决策 + 反馈闭环。你的审核链路天然是多步决策场景。
  • 基础：ReAct vs Plan-and-Execute、Function Calling 原理、Agent vs Prompt Chain 本质区别
  • 记忆设计：工作记忆 / 会话记忆（摘要滚动）/ 长期记忆（向量+结构化）三层架构
  • 多智能体：Multi-Agent 协作、A2A 框架、MCP 协议、框架选型（LangGraph/LlamaIndex）
  • 工程化：Agent 评估指标、失败模式、可控性与对齐
  • 📌 复习锚点：agent-architecture · react-and-planning · mcp-protocol
4
微调 + 推理优化期 · 工程分水岭 约 4 周 ★拉差距
微调题从 SFT 扩到 DPO/LoRA/QLoRA，核心考 trade-off 不是公式。推理优化是 2026 最能拉开差距的一块——很多人模型能跑起来，一问线上服务就虚。
  • 微调：SFT/RLHF/DPO/GRPO 区别、LoRA rank 影响、QLoRA 低显存原理、何时不该微调
  • 推理优化五层：模型层（量化/蒸馏/小模型路由）· 结构层（KV Cache/MQA/GQA/投机解码）· 算子层（FlashAttention）· 服务层（连续批处理/PagedAttention/Prefix Cache）· 业务层（缓存/降级/上下文裁剪）
  • 部署：vLLM / Ollama / TensorRT-LLM 选型、Prefill vs Decode、TTFT/TBT/吞吐/P99 指标
  • 📌 复习锚点：lora-and-peft · dpo-grpo-alignment · precision-and-quantization · small-language-models-on-device
5
系统设计 + 项目 + 面试期 · 收口上岸 约 4 周 ★决定 offer
二面必考系统设计：「设计支持百万文档的 RAG 系统」「设计企业智能客服」。要有 2–3 个能写简历、能口述、可演示的项目。
  • 系统设计：高并发 LLM 后端架构（FastAPI+vLLM+Redis）、限流/降级/监控、成本控制
  • 项目（直接写简历）：① 审核 RAG 知识库 ② 审核 Agent 助手 ③ 多模态审核 Pipeline（结合你的漫剧业务）
  • 面试：Java+AI 双语言八股、项目亮点包装、口述训练
  • 📌 复习锚点：fastapi-audit-service · springboot-integration · llm-as-judge-and-red-teaming
