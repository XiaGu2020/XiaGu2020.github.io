🆕 2026 必补的新热点（旧资料里没有，最近面经新增）

① Context Engineering（上下文工程）— 比 Prompt 工程更上一层
2026 Agent 面试的新核心。Prompt 工程是「怎么写好一句指令」，Context Engineering 是「怎么管理整个上下文窗口」：System Prompt + 最近 N 轮 + 当前任务目标 + 可用工具结果，按语义重要性裁剪而非简单截断。长任务里 Agent 跑偏、自我绕进死循环，根因往往是 Context 没管好。要点：State Schema（TypedDict 严格状态）、Trim Strategy（语义保留）、Summary Buffer（旧对话摘要化塞回头部）。
② 模型网关 Model Gateway — P8/P9 系统设计高频
企业级 AI 平台的中枢。屏蔽不同模型供应商（OpenAI/Anthropic/Qwen/DeepSeek）的协议差异，统一 API + 热插拔；基于延迟/成本/吞吐/优先级动态路由到最优后端；网关层做 Guardrails（PII 脱敏、Prompt 注入防御、违规词拦截）、语义缓存（拦截高频相似请求降本）、多租户 Token 计费配额。Java 选手优势：Spring Cloud Gateway 可直接迁移这套设计。
③ LLMOps / AgentOps 全生命周期 — 上专家级的门票
从「会用模型」到「能稳定交付到业务」。LLMOps：数据→微调→评测→推理→上线→监控→迭代闭环；AgentOps：Agent 构建→评测→调优→可观测。可观测三件套：TTFT/TPS/成功率监控 + 端到端 Trace（Prompt→工具调用→输出，类 LangSmith/Phoenix）+ token 利用率/幻觉率健康看板。这是 P8/P9 JD 的硬门槛（要求 1 年以上 MLOps/LLMOps 经验）。
④ 质量闭环 + 风险治理 — 二面「上线后变差怎么定位」必问
质量闭环：Golden Set（黄金评测集）+ LLM-as-Judge + 线上灰度 + Trace 回放 + 人工复核怎么配合。风险治理：Prompt 注入、工具越权、隐私泄露、RAG 权限过滤、模型供应商限流故障降级。高频追问：「如何证明新 Prompt 比旧 Prompt 更好」「供应商限流怎么降级」「Agent 工具调错怎么办」——都要给工程闭环答案，不是概念。
⑤ 多模态审核 SOTA（你的方向，必须跟踪）
腾讯云 VM：视觉+语音+OCR 三模态协同，主打高并发、低延迟、高可用（厂商宣称万级 QPS / 毫秒响应 / 99.9% 可用，面试引用时说「高并发高可用」即可，别背死具体数字）。快手 BLM-Guard（AAAI 2026）：多模态 CoT（视觉定位→风险筛查→因果分析→判决）+ 策略对齐 RL（GRPO + 自适应 critique reward），解决 VLM 黑盒、策略漂移问题。趋势：数据飞轮驱动模型长效进化、可解释审计（注意力热力图 + token 溯源）、深伪武器化对抗。这块是你简历项目的天花板模板。
